# 費用について

## 学費には何が含まれますか？

### DTS講義期間 / セカンダリー・スクール

3ヶ月間のDTS講義期間、またはセカンダリー・スクール（DBSなど）にかかる費用は以下の通りです：

**授業料：300,000円（税込）**
これには講義代、事務手数料、リフレッシュメント（軽食・飲み物）が含まれます。

**DTS+DBS セット割引：** 両方のスクールに申し込む場合、DBSの授業料から**50,000円の割引**が適用されます（DTSを無事に修了することが条件となります）。

**宿泊費：120,000円**
これには寝具、光熱費、Wi-Fi利用料が含まれます。

**食費、現地交通費、個人諸費：予算目安 150,000円**
共同での食事（費用への寄付をお願いする形になります）、外食、各自で用意する時間が混在します。また、ミニストリーや教会へ向かう際の電車賃も必要です。その他、個人の雑費として別途お金が必要になる場合もあります。この金額は推定予算です。

### アウトリーチ期間

アウトリーチ期間を含むコース（DTSなど）については、予定されている費用を事前にお伝えします。費用はアウトリーチの場所や期間によって異なります。

**推定予算：400,000円**
通常、これには交通費、主要な食事代、宿泊費、およびその他の必要経費が含まれます。個人の諸経費や予防接種の費用は、提示されるアウトリーチ費用には含まれません。

### DTS 合計費用

念のためお伝えしますが、DTSを修了するには講義期間とアウトリーチ期間の両方を完了する必要があります。そのため、**スクール期間全体で約970,000円の予算を立てておいてください**。

## 支払い方法を教えてください。

授業料の支払い方法の詳細は、[支払いページ](../about/payment.md)でご確認いただけます。

**宿泊費、食費、およびアウトリーチ費用は「現金のみ」での支払いとなりますのでご注意ください。**

## 支払期限はいつですか？

### DTS講義期間 / セカンダリー・スクール

- 合格後、席を確保するために100,000円のデポジット（予約金）の支払いが必要です。
- 残りの授業料200,000円は、到着日の10日前までに支払う必要があります。
- 宿泊費120,000円と、最初の1ヶ月分の共同食事費は、到着時に現金で支払う必要があります。
- その後の共同食事費は、第4週と第8週の初めに支払います。

### アウトリーチ期間

- アウトリーチ出発の1ヶ月前までに、150,000円のデポジットが必要です。
- 残金はアウトリーチ出発の2週間前までに支払う必要があります。

## 返金規定はどうなっていますか？

### DTS講義期間 / セカンダリー・スクール

原則として、一度支払われた授業料は返金されません。事情によっては相談に応じる場合があります。宿泊費と食事代については、週単位で日割り計算して返金されます。

### アウトリーチ期間

- アウトリーチ開始前：支払済みの全額から、航空会社のキャンセル料およびすでにアウトリーチのために支払われた経費を差し引いた額を返金します。
- アウトリーチ期間中：宿泊費と食事代を週単位で日割り計算して返金します。*ただし、航空会社の調整金、チケット変更手数料、およびその他の返金不可の経費を差し引いた額となります。*









# 費用

受け入れられた場合、以下の費用にご留意ください（変更になる可能性があります）。

## 3ヶ月間のDTS講義期間／3ヶ月間のDBS

- **授業料：**¥300,000 (税込)。講義、運営費、軽食が含まれます。
- **宿泊費：**3ヶ月間で ¥120,000。
- **食費と現地交通費：**別途 ¥150,000を見込んでください。

合計で、3ヶ月間の総予算は **¥570,000程度**となります。

## DTSアウトリーチ期間および総費用
DTSの場合、必要な2ヶ月間のアウトリーチ期間の費用は、渡航先によって異なり、別途かかります。**約 ¥400,000**を見込んでください。

したがって、**DTS全体（5ヶ月間）の総費用は、約 ¥970,000となる見込みです**。

## DTS+DBS COMBO割引

DTSとDBSの両方の学校に申し込む場合、DBSの授業料から **¥50,000の割引**が適用されます。
